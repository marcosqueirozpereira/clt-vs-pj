# ⚖️ Comparador CLT x PJ

Uma calculadora online para comparar remuneração **CLT x PJ** de forma simples.

## O que este app faz

Você informa:

- salário CLT
- benefícios
- descontos
- faturamento PJ
- impostos
- contador
- plano de saúde
- reservas de férias, 13º e FGTS

E o app mostra:

- líquido mensal CLT
- poder de compra CLT
- líquido mensal PJ
- líquido anual
- diferença real entre os dois cenários

pip install -r requirements.txt
